## 📝 제출 방식

1. **GitHub 저장소 생성**: 과제 코드를 GitHub 저장소에 업로드
2. **이메일 제출**: marin@medisolveai.com 으로 아래 형식에 맞춰 제출

### 제출 항목 (필수)

GitHub 저장소에 다음 항목들이 반드시 포함되어야 합니다:

1. **소스 코드**
   - 모든 API 모듈 코드 (Gateway, Patient API, Admin API)
   - 설정 파일 및 환경 분리 관련 파일

2. **데이터베이스 마이그레이션 파일**
   - 스키마 생성 SQL 파일 (init.sql 등)
   - 또는 Alembic 등 마이그레이션 도구 파일

3. **테스트 데이터셋 SQL 파일**
   - 개발/테스트 시 사용할 샘플 데이터 SQL 파일
   - 의사, 진료 항목, 환자, 예약 등의 초기 데이터 포함

4. **README.md**
   - 프로젝트 설명
   - **실행 방법** 
     - 환경 설정 방법
     - Docker Compose 실행 명령어
     - 테스트 실행 방법
     - API 접근 방법 (포트, 엔드포인트 등)
   - **AI 활용 방식**

5. **Docker 관련 파일**
   - docker-compose.yml
   - Dockerfile (필요 시)
